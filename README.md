# کۆدەنگ | KODANG

وێبسایتێکی هەواڵی مۆدێرن بە Next.js و TypeScript دروستکراوە.

## تایبەتمەندیەکان | Features

- ✨ UI/UX مۆدێرن و جوان
- 📱 پشتگیری ئامێرە جیاوازەکان (Responsive)
- 🌍 پشتگیری چەند زمان (کوردی، فارسی، ئینگلیزی)
- ⚡ خێرایی و کارایی بەرز
- 🔍 سیستەمی فیلتەرکردن بەپێی بابەت
- 📰 پەڕەی وردەکاری هەواڵەکان

## دەستپێکردن | Getting Started

### پێویستەکان | Requirements

- Node.js 18+ 
- npm or yarn

### دامەزراندن | Installation

```bash
# دامەزراندنی پاکێجەکان
npm install

# دەستپێکردنی سێرڤەری گەشەپێدان
npm run dev
```

وێبسایتەکە لە [http://localhost:3000](http://localhost:3000) دەکرێتەوە.

## پێکهاتەکان | Structure

